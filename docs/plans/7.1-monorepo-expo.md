# Plan: Monorepo + Expo App

## Context

Odlingsguiden är live som webbapp. Nu bygger vi en React Native-app med samma data och design. Istället för en hack-lösning med `watchFolders` gör vi det rätt: monorepo med npm workspaces där delad kod är ett explicit paket.

## Monorepo-struktur

```
odlingsguiden/
  package.json                      # Root: workspaces, gemensamma scripts
  tsconfig.base.json                # Delade TS-inställningar

  packages/
```

## Steg

### Fas 1: Monorepo-omstrukturering

#### 1.1 Feature branch
- `git checkout -b feature/monorepo-expo`

#### 1.2 Root workspace
- Skapa root `package.json` med `"workspaces": ["packages/*"]`
- Skapa `tsconfig.base.json` med gemensamma TS-inställningar

#### 1.3 Shared-paket (`packages/shared/`)
- Flytta `src/data/types.ts`, `zones.ts`, `categories.ts`, `difficulty.ts`, `cropList.ts`, `cropSlugs.ts`, `cropUtils.ts` → `packages/shared/src/data/`
- Flytta `src/data/crops/*.ts` (85 filer, INTE index.ts) → `packages/shared/src/data/crops/`
- Flytta `src/utils/monthParser.ts`, `slug.ts` → `packages/shared/src/utils/`
- Flytta `src/utils/seasonData.ts` → `packages/shared/src/utils/` (refaktorera: exportera sync-version som tar crops som parameter, ta bort import av loadAllCrops)
- Skapa `packages/shared/src/theme/colors.ts` — extrahera `colors` från `src/theme/tokens.ts`
- Skapa `packages/shared/src/theme/tokens.ts` — plattformsoberoende tokens med nummer (spacing: 4, 8, 12... fontSize: 11, 13, 15... radii: 12, 16, 20)
- Skapa `packages/shared/src/index.ts` — barrel export
- Skapa `packages/shared/package.json` (`"name": "@odlingsguiden/shared"`, `"main": "src/index.ts"`)

#### 1.4 Web-paket (`packages/web/`)
- Flytta nuvarande `src/` → `packages/web/src/` (exklusive det som gick till shared)
- Flytta `vite.config.ts`, `index.html`, `tsconfig.json` → `packages/web/`
- Flytta `scripts/`, `server/`, `dist/`, `public/`, `nginx.conf` → `packages/web/`
- Uppdatera alla imports: `../../data/types` → `@odlingsguiden/shared`
- Behåll `packages/web/src/data/crops/index.ts` (Vite-specifik loader)
- Uppdatera `packages/web/src/theme/` — importera `colors` + `tokens` från shared, behåll `global.css` med CSS custom properties
- Uppdatera `vite.config.ts` — samma config, justerade paths
- Verifiera: `npm run dev` i web/ fungerar som vanligt

#### 1.5 Verifiera webb
- `cd packages/web && npm run dev` — allt ska fungera
- `npm run build && npm run prerender` — 102 sidor
- Alla existerande tester/linting passerar

### Fas 2: Expo-app

#### 2.1 Scaffold
- `npx create-expo-app@latest packages/app --template blank-typescript`
- Deps: `expo-router`, `react-native-svg`, `@react-native-async-storage/async-storage`, `expo-font`
- `package.json`: `"dependencies": { "@odlingsguiden/shared": "*" }`
- `tsconfig.json` med paths om Metro behöver det
- `metro.config.js` — symlink-stöd för workspaces

#### 2.2 Data layer
- `packages/app/src/data/crops.ts` — statisk import av alla 85 grödor från `@odlingsguiden/shared`
- Exporterar: `getCachedCrop()`, `loadCrop()`, `loadAllCrops()`

#### 2.3 Theme
- `packages/app/src/theme/styles.ts` — importerar tokens från `@odlingsguiden/shared`
- Ladda Fraunces + Lora via `expo-font`

#### 2.4 Ikoner & illustrationer
- Konvertera alla 85 illustrationer till react-native-svg
- Konvertera alla ikoner
- `CropIcon.tsx` + `CropGraphic.tsx` med synkron lookup

#### 2.5 Screens
- **CropListScreen**: FlatList, kategori-flikar, sök, zon-pill
- **CropDetailScreen**: ScrollView med alla sektioner
- **SeasonScreen**: månadsnavigering, aktivitetsgrupper
- **ZoneSelector**: modal med 8 zoner, AsyncStorage

## Tokens-strategi

Shared-paketet definierar plattformsoberoende tokens:
```ts
// packages/shared/src/theme/tokens.ts
export const spacing = { xs: 4, sm: 8, md: 12, lg: 16, xl: 20, xxl: 24, section: 40 }
export const fontSize = { small: 11, badge: 12, label: 12, body: 13, h2: 15, dataLarge: 22, h1: 28 }
export const radii = { cardSmall: 12, button: 14, card: 16, cardLarge: 20, badge: 20 }
export const fontWeight = { regular: '400', semibold: '600', bold: '700', extrabold: '800' }
```

Webb mappar till CSS: `global.css` läser tokens → custom properties.
App använder direkt i StyleSheet.

## Tekniska noteringar
- `seasonData.ts` refaktoreras: sync-version tar `allCrops` som parameter (redan finns `getSeasonActivitiesSync`). Webb-wrappern anropar `loadAllCrops()` → sync. App-wrappern gör samma med sin loader.
- Metro i Expo behöver konfigureras för npm workspaces (`watchFolders` för root `node_modules`, symlink support)
- Inga charts i första version (Recharts → victory-native görs separat)
- `ssrSeasonCache.ts` stannar i web/ (SSR-specifik)
- `dist/` stannar i `packages/web/dist/`
